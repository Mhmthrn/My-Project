Navigate the DOM Tree with a keyboard

Elements ekraninda acilan bolumde klavye tuslarimizla gezebilir, sag-sol, yukari ve asagi yonlerinde hareketle kodlari acip veya kapatabiliriz.

Scroll Into View

Gorus alanimizin disinda kalan bir koda ihiyac duydugumuzda okoda hizlica erisim zapabilmemize olanak tani.

Search For Nodes

Elements paneline geldigimizde Control + F veya Command + F (Mac) tuşlarına bastigimizda DOM Ağacının en altında Arama çubuğuaçılır. Bu bolum kodlar arasinda gezinmemize ve arama yapmamiza imkan verir.

2

Edit the DOM

Edit Content

DOM Ağacı'ndaki içeriğe çift tıkladigimizda ilgili kod uzerinde degisiklik yapmamiza ve yapilan degisikligi aninda gormemize imkan verir.

Edit Attributes

Secmis oldugumuz kodun niteliklerini degistirebiliriz. Örnegin yazinin arka plan rengi, vurgusu vb. ozelliklerini.

Edit Node Type

Sectigimiz kodun turunu degistirebiliriz. Örnegin li olan bir siralama kodunu input sekline cevirebiliriz.

Reorder DOM Nodes

Sectigimiz kodun sirasini degistirebiliriz.

Force State

Sectigimiz kodun :active, :hover, :focus, :visited, and :focus-within ile vurgulanmasi veya farkli ozellikler ekleyebiliriz degistirebiliriz.

Hide a Node

Sectigimiz kodun uzerine geli H tusuna basarsak ilgili kod gozukmez ve dom agacinda sol tarafinda gri yuvarlak bir ikon belirir. Gizliligi tekrar H tusuna basarak iptal edebiliriz.

Delete a Node

Sectigimiz kodusilebiliriz devaminda geri almak icin Control+Z or Command+Z (Mac) tuslarian basarak islem yapariz.
